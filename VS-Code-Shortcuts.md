# VS Code Shortcuts

## Line

|    Key     |        Function        |
| :--------: | :--------------------: |
|     ⌘C     | Copy (Empty Selection) |
|     ⌘X     | Cut (Empty Selection)  |
|    ⇧⌘K     |         Delete         |
|  ⇧⌘Enter   |      Insert Above      |
| ⌥ Up/Down  |      Swap Up/Down      |
| ⇧⌥ Up/Down |   Dupulicate Up/Down   |


### Cursors

|   Key   |             Function             |
| :-----: | :------------------------------: |
|   ⌘D    | Add Selection to Next Find Match |
| ⌥ Click |            Add Cursor            |

### Moves

|     Key      |          Function           |
| :----------: | :-------------------------: |
| ⌘ Left/Right | Go to Beginning/End of Line |
|  ⌘ Up/Down   | Go to Beginning/End of File |
| ⌥ Left/Right |        Move by word         |

### Code Management

|  Key  |        Function         |
| :---: | :---------------------: |
| ⌘KO/J | Fold/Unfold All Methods |

### Display

|  Key  |      Function      |
| :---: | :----------------: |
|  ⇧⌘\  | Change to Next Tab |
|  ⌘B   |   Toggle Sidebar   |
|  ⌃⇧`  |    New Terminal    |
|  ⌘J   |     Show panel     |

### File

|  Key  |  Function   |
| :---: | :---------: |
| ⌘ Del | Delete File |

### MarkDown Shortcuts

|  Key  |          Function          |
| :---: | :------------------------: |
| ⌘K, V |  Open preview to the Side  |
|  ⌘⇧V  |        Open preview        |
|  ⌃⇧S  | Sync preview / Sync source |
|  Esc  |     Toggle sidebar TOC     |
